Route HTTP request to the appropriate handler.

Parses the HTTP request line to determine the requested endpoint and dispatches to the corresponding handler function.

Supported endpoints: GET /start-recording - Start saving frames to disk GET /start-live - Live MJPEG stream GET /playback - Stream from saved recording

pipelinePipeline context containing camera, buffer, and detector state
